Big Data

Como a Ciência dos Dados transformará a Sociedade como a conhecemos

Descrição

A cada dia são criados mais de um quintilhão de dados, de modo que 90% dos dados existentes hoje foram criados nos últimos dois anos. Esses dados vem de muitos lugares diferentes, como os tradicionais sistemas Web, mas também de sensores usados para obter informações climáticas, mensagens postadas em redes sociais, imagens digitais, vídeos, entre outras. Esses dados são chamados de Big Data. A gestão eficiente da enorme quantidade de dados dos sensores está se tornando cada vez mais importante. Algumas técnicas de agregação, pré-processamento, modelagem e processamento de dados estão sendo exploradas para solucionar este problema e representam o início da transformação da sociedade como a conhecemos.
